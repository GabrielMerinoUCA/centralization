VII. New string methods

Javascript added a bunch of new methods including string methods. This methods can be useful tools in certain scenarios so it could be worth being aware of them.

7.1. Repeat method

The first string method is called "repeat". It uses the assigned string and returns copies of it which numbers we can specify

main.js



let myString = "What ";

// will print "What What What What What"

console.log( myString.repeat(5) );

7.2. startsWith method

The "startsWith" method checks if the string starts with a specific sequence of characters from a specific index and returns a boolean with the corresponding validation

main.js



let myString = "goodbye";

/*
will print "true"
arg1: string to validate
arg2: index to start from (first is 0)
*/

console.log( myString.startsWith( "good", 4 ) );

7.3. endsWith method

Similarly to "startsWith", "endsWith" is a method that checks if the string ends with a specific sequence of characters from a specific index and if it does, will return a boolean with the corresponding validation.

main.js



let myString = "goodbye";

/*
will print "true"
arg1: string to validate
arg2: index to start from (first is 0)
*/

console.log( myString.endsWith( "good", 4 ) );

7.4. include method

The "include" method checks if the sent string exists in the main string.

main.js



let myString = "My name is Jeff";

// will print "true"

console.log( myString.includes("name") );
