VI. Object literals enhancements

We know that in Javascript, object literals have the ability to pair their keys with an external variable and automatically reference the value of such variable as well as the ability to contain methods within. However, the old syntax is quite antiquated and there is a better way of doing it.

main.js



let name = "John";

let belt = "white";

let ninja = {

 // Doesn't require "name: name" anymore but is inferred
 name, belt,
 // Doesn't require assignation of anonymous function anymore
 chop(x) {
   console.log(`Chopped the enemy ${x} times!`);
 }

}
