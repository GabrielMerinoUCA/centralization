II. Constants

We know what constants are. Just to give a refresh on when and how to use them, just remember:

  1. Make them global
  2. Make them descriptive
  3. Name them using all caps with snake case (MY_CONST)
  4. Use them to avoid magic numbers (like 3.14, what does that mean? well of course, 'pi')

main.js



const PI = 3.14;

// Naturally, if you try to assign a new value after the declaration, JS will return an error

Be careful since as we know by now, JS always has some funky business to watch out for

const PI = 3.14;

function calcArea(r) {

 const PI = 10;
 console.log( PI * r * r);

}

// Will return 250 instead of the expected value

calcArea(10);
