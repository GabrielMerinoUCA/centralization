IV. Spread operator

The spread operator allows us to extract the elements of an array into a single entity.

main.js



let array = ["John", "Monroe", "Doever"];

// Will print "John Monroe Doever" instead of ["John", "Monroe", "Doever"]

console.log(...array );

The spread operator can be useful whenever you want to add an array into another array without it storing the "Array" as an item but instead the individual items.

main.js



let nums1 = [1,2,3];

let nums2 = [...nums1,4,5,6];

// will print 1 2 3 4 5 6 instead of [Array, 4, 5, 6]

console.log(nums2 );

Another application of the spread operator is to use an array directly on a function that expects multiple arguments

main.js



let array = [3,5,7];

function addNums(a,b,c) {

 console.log(a+b+c);

}

// the array will spread on 3 items that will be passed to the corresponding argument

addNums(...array)
