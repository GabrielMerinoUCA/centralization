V. Template Strings

Template strings are special type of writing strings that allows for line breaks inside the string to be accounted without the need for a break character and also allow us to interpolate variables without the need for concatenation.

main.js



function logUser(name, age) {

 console.log(
  `My name is ${name}
  and my age is ${age}`
 );

}

// Will print:

/* My name is John
and my age is 28*/

logUser("John", 28);
