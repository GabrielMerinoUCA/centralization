VIII. Arrow functions

Arrow function is a new way to create anonymous functions in Javascript.

main.js



let myFunc = () => {

 console.log("Hello world");

}


// Prints hello world

myFunc();


Using arrow functions brings an advantage when compared to traditional anonymous functions. For instance, given the following code:

main.js



let user = {

 name: "John",
 chop(x) {
  window.setInterval(function() {
   if(x>5) {
    // won't print name because "this" is referencing the function object and not the object literal
    console.log(`${this.name} chopped ${x} times`);
    x--;
   }
  });
 }

The arrow function solves this issue without further complications

main.js



let user = {

 name: "John",
 chop(x) {
  window.setInterval(() => {
   if(x>5) {
    // Will now reference the object literal thus printing the name
    console.log(`${this.name} chopped ${x} times`);
    x--;
   }
  });
 }
