I. Introduction

We all hear these buzzword 'ECMAscript' but what is it actually? ECMAscript is essentially a standard created by ECMA international to standardize script languages used on the web. Javascript is just an implementation of such standard.

This tutorial will focus on ES6. However, ES6 by the time I'm writing this (2023) is outdated in the sense that the features are very common in current js implementations/projects; that's why it is important to know such features.

Also, this course will only cover the most notable features that were introduced on ES6 so this course will be short-lived.
