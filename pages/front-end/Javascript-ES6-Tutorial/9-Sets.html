IX. Sets

9.1. Basics of the Set structure

Sets are a data structure similar to arrays that prevents storage of duplicates

main.js



let mySet = new Set();

mySet.add("John").add("Rick").add("John");


// Will print "John", "Rick" without repeating John a second time

console.log(mySet);


// Just like any other data structure, there is a method to visualize the amount of elements inside it

console.log(mySet.size);// prints 2


// If there is the ability to add, then there also is the ability to remove. The method returns a bool btw

mySet.delete("John");


// Will now only print "Rick"

console.log(mySet);


// This method checks if an element exists within the set

mySet.has("Rick");// will return true;


// Delete all contents of the set

mySet.clear();


9.2. Convert arrays with duplicates to set without duplicates

It's possible to convert a massive array of items that might have duplicated items into a set that removes such duplicates

main.js



let array = [ "John", "Rick", "Doever", "Pepe", "John", "Doever" ];

let mySet = new Set(array);


// Prints "John", "Rick", "Doever", "Pepe"

console.log(mySet);
