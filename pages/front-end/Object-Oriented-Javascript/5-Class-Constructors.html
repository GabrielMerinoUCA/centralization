V. Class Constructors

Constructors in Javascript are a little bit different than some languages like Java, C or C#. Instead of using the name of the class itself as the constructor, we use the 'constructor' keyword. In addition to the 'constructor' keyword being an oddity, Javascript also is only capable of defining attributes inside such keyword instead of outside like many other programming languages.

main.js



class User {
     constructor(email, name) {
          this.email = email;
          this.name = name;
     }
}


let user1 = new User("john@gmail.com", "John Doe");

let user2 = new User("bob@gmail.com", "Bob Dom");
