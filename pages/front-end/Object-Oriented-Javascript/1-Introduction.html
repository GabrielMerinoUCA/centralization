I. Introduction

On Javascript, almost everything is an object, from arrays to strings and even primitives since there are some methods that we can use on numbers for example. We already know what objects are but for quick refresh, an object is an abstraction of a concept or real life object and it has properties and methods.

Even if everything might appear to be an object, there some things that are not considered objects like for example, the "null" value.
