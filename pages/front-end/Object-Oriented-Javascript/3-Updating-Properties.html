III. Updating Properties

Since Javascript doesn't have private keywords, you can directly access the attributes and change them by calling them

main.js



let userOne = {
     email: "john@gmail.com",
     name: "John Doe",
     login: function() {
          console.log(this.email);
     }
}

userOne.name = "Bob Johnson";


There is also another way of updating attribute on Javascript

main.js




. . .

/* You can access their value as if they were associative arrays, which makes sense since objects in json behave like that and json is based on Javascript's objects*/

userOne["email"] = "bob@gmail.com";


// There is also other advantages such as selecting an attribute dynamically

let prop = "name";

userOne[prop] = "Micheal Dom";
