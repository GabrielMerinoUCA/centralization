II. Object Literals

We've already seen object literals on the Javascript Tutorials for Beginners (Chapter 27). We just called them objects. Here's a quick refresh on the syntax:

main.js



let userOne = {
     email: "john@gmail.com",
     name: "John Doe",
     login: function() {
          console.log(this.email);
     }
}

console.log(userOne.name);

userOne.login();
