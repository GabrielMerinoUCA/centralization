XXXI. What is the DOM in Javascript

We've heard of the DOM before. We know it stands for Document (HTML) Object (every element is an object) Model (a node-like representation of the document hierarchy), but what does it mean? Without getting into to much technicality, the DOM is a representation of a markup document such as HTML that can interact directly with such document.This is important as if we want to make changes to the html with Javascript, we can do it use something like the DOM API. In other words, the DOM is a mean in which we can interact with, to manipulate the html using Javascript.
