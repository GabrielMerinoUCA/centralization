III. Hello World! in Javascript


3.1 Embedded Javascript

Embedded javascript is essentially adding you javascript directly into the html page using a script tag.

It is a common practice to put embedded scripts (and scripts in general) at the bottom of the body element. This allows the html to load first before the JS code executes; which can take time that could be instead spent loading the content so the user doesn't think the site is down or anything like that.

This also lines up with the principle of not depending on JS for core functionality as if the JS code is loading last and presents core functionality, the html won't work properly or at all.


3.2 Hello World!


index.html



<body>


     . . .

     <!-- Always at the bottom -->

     <script>

          alert("Hello World!");

     </script>

</body>
