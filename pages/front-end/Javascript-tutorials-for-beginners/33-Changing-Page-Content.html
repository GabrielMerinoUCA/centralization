XXXIII. Changing Page Content

33.1. innerHTML

In order to change the content of a web page, first we have to grab an element and then, we have to the prebuilt methods to change the content.

index.html



<body>

    . . .

</body>


test.js



let myBodyTag = document.getElementsByTagName("body");

console.log(myBodyTag[0].innerHTML); // Will print the whole document on it's basic html form


/*Because you now have a reference to all the contents on the body tag, you can change all that's inside of it using the property*/

myBodyTag[0].innerHTML = "<p>Hello world</p>"; // overwrites the whole page with the content

33.2. textContent

innerHTML is usually used when you want to write html into a specific element. That is useful in some scenarios but not all of them. If you want to see the plain text of the element in question or just change the inner text of it, there is a different property.

index.html



<body>

    <h1>

        . . .

    </h1>

</body>


test.js



let myHeaderTag = document.getElementsByTagName("h1");


/* Prints only the inner text of the element (same as innerHTML but without the tags) */

console.log(myHeaderTag[0].contentText);


/* Because we are changing the text, the html will show "<p>Hello</p>" literally, with the tags*/

myHeaderTag[0].contentText = "<p>Hello</p>"
