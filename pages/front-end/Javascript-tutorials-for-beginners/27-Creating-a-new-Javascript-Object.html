XXVII. Creating a new Javascript Object

27.1. Adding attributes to Object

Javascript has some uniqueness with objects. Unlike other languages that force you to define a class so you can instantiate an object, Javascript allows you to insert attributes on a variable after it's definition with no problem.

test.js



// Specifying "new Object()" is obligatory

let myCar = new Object();


myCar.topSpeed = 75;

myCar.driver = "John";


console.log( myCar.drive );// Prints John

27.2. Adding methods to Object

Asides from attributes, you can also insert methods.

test.js



let myCar = new Object();


/* Javascript allows assignation of functions to variables/attributes */

myCar.drive = function() {
    console.log( "Driving" );
};

myCar.drive(); // Prints "Driving"

27.3. The short handed version of creating objects

The same way you can immediately assign an array a list of values, you can assign a whole pre-defined object structure to a variable.

test.js



let myCar2 = {
    // attribute: value
    topSpeed: 115;
    driver: "John"
    // method: function
    drive: function() {
        console.log( "Driving" );
    }
 };
