XIV. Logical Operators

14.1. AND & OR

We know logical operators, they are used to check multiple conditions.

test.js



var myAge = 19;


if(myAge > 18 && myAge <= 30) {
    console.log( "You are allowed on the club" );
}

else{
    console.log( "You are not allowed on the club" );
}

Of course we also have the OR operator.

test.js



var myAge = 20;

if(myAge <= 18 || myAge >= 60) {
    console.log( "You are not on the working age" );
}

else {
    console.log( "'A chambear boludo!'" );
}
