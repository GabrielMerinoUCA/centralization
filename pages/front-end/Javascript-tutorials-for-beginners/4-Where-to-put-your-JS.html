IV. Where to put your JS


4.1. JS scripts on separate files

If a script is longer than 10 lines, put it on a separate .js file, here's how you do it.


index.html



<body>


     . . .

     <!-- Referencing external .js file -->

     <script src="./test.js"></script>

</body>


Then add a file ending with the .js extension


test.js



alert("Hello world");
