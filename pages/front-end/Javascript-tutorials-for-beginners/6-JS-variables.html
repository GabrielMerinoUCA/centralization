VI. Javascript variables


6.1. Variables and data types

Variables are declared using the "var" keyword. Just like any other programming language, Javascript has the typical primitive data types (int, string, booleans, etc...)

var myVar = 5;


Javascript variable's datatype is assigned implicitly, which means that their datatype depends on the value assigned to them. Additionally, you can leave a variable unassigned and give it a value later. (however, it's not a good practice!)

var myVar;

myVar = 5;


Asides from ints, Javascript supports other data types like strings, which can be stated by using either double or single quotes.

// Both ways are valid

var myString1 = "hello"

var myString2 = 'hello again'


Javascript variables are loosely typed, which means that their datatype can change. However, this is not a good practice so don't do it

var myVar = 5;

myVar = "Hello"; // this works
