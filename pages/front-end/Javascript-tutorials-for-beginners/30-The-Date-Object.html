XXX. The Date Object

30.1. The Date Constructor

The Date object is very common in multiple programming languages. Javascript has a very similar implementation as all of them.

test.js



let myDate = new Date();


console.log(myDate); // Prints the current date


/* You can also specify any date
arg1: year
arg2: month (0-11)
arg3: date
arg4: hour (0-23)
arg5: minute (0-59)
arg6: second (0-59)
*/

let pastDate = new Date(2000, 5, 16, 13, 15, 0);

console.log(pastDate); // Prints "16 June 2000, 1:15 PM"

30.2. Date Methods

Since Date is an object and we know Javascript prebuilt features have methods and other properties within, we will talk about some of Date's methods.

test.js



let myDate = new Date(2000, 5, 16, 13, 15, 0);


// Prints the month (0-11)

console.log(myDate.getMonth());


// Prints the year

console.log(myDate.getYear());


// Prints the date (1-31)

console.log(myDate.getDate());


// Prints day of week (0-6)

console.log(myDate.getDay());


// Prints the hour

console.log(myDate.getHours());


// Prints number of milliseconds since 1st Jan 1970

console.log(myDate.getTime());

/* Javascript is weird, but we already know that. A very useful reason for "getTime()" to exist is to compare since even if two Dates are the exact same, you won't be able to compare them using the "==" operator. */
