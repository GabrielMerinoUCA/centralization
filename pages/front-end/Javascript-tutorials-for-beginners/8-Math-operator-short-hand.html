VIII. Math Operator Short-hand


8.1. Basic short-handed operators

Last tutorial I was wondering if we were going to talk about some handy operators found in other languages like += operator. Apparently, this chapter is for that Ok so what is the += operator anyways? This operator allows for a variable to be equal to the value held on itself plus another number

var myVar = 5;

myVar += 5; // myVar will be 10, is the same as saying myVar = myVar + 5;


Asides from the += operator, there are other so called "short-hand" operators for most arithmetic operations that work in same fashion.

var myVar = 4;


// Subtraction variation

myVar -= 2; // myVar = 2


// Multiplication variation

myVar *= 10; // myVar = 20


// Division variation

myVar /= 5; // myVar = 4

8.2. Even more short-handed operators

We know this operator from other languages, is the ++ and -- operator. They essentially are short-handed operators from the short-hand operators +=1 or -=1.

var myVar = 1;


// Addition operator

myVar++; // myVar = 2

++myVar; // myVar = 3


// Subtraction operator

myVar--; // myVar = 2

--myVar; // myVar = 1

Remember that depending on where you put the operator matters, if you put it before the variable name, the value of the variable will change before being used. If you put the operator after the variable, then the value of the variable will be used and then will be changed so be careful; usually it's common practice to just use it before the variable.
