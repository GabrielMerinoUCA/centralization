XLI. Javascript Timers

41.1. setTimeout

You might have seen some websites that have images that change after some time. Those were made using Javascript.

index.html



<body>

     <p style= "opacity: 0;" id= "text-1" >Showing text 1 </p>

</body>


test.js



let text1 = document.getElementById( "text-1" );

let showTag = function() {
     test1.style.opacity = "1";
}


/* execute sent function after specific time
arg1: function to be executed
arg2: Time in milliseconds to wait before executing such function
*/

setTimeout(showTag, 3000);

41.2. setInterval

I previously referred to some changing images on websites as action of Javascript. However, I did not introduce a clear way to replicate such effect. Yes you could theoretically make a loop on the setTimeout but why do that when Javascript already has a prebuilt function.

     <div class= "100px-square" id= "square-1" >Square 1 </p>

</body>
