XVI. For Loops

Again, very common feature, we're not going with anything fancy like kotlin's forced foreach loops, we have the option of going with normal for loops.

test.js



for(var age = 5; age < 18; age++) {
    console.log( "You are underaged" );
}

console.log("You are now an adult");


By the way, Javascript doesn't have foreach loops like on Java or C#. If you want to use a foreach loop, you'll have to use it directly on arrays as a method.
