XXII. NaN (Not a Number)

22.1. Nothing makes sense

Now we're getting into the weirdness of Javascript. Remember what I said about arithmetic operators not working with the combination of string and numbers on chapter VII? Well, I was wrong and correct. You see, you cannot do arithmetic operations between a string and a number UNLESS the string is made entirely of a number.

test.js



console.log( "h" / 5 );// Prints NaN

console.log( "5" / 5 );// Prints 1


So in other words, the only arithmetic operator that doesn't work is the addition operator.

22.2. Check NaNs

If you want to check that an argument or variable is NaN, you have to use another Javascript prebuilt function called isNaN(). And even though NaN is value that exists, just saying x == NaN won't work.

test.js



let a = "a";

let b = 5;


if(isNaN(a)) {
    console.log( "'a' is a NaN" );
} else{
    console.log( "This console.log() wont' be showed" );
}
