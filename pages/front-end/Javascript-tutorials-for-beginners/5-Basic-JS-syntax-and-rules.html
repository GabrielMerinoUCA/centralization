V. Basic Javascript syntax and rules


So JS is like any other languages, it follows similar syntax and some unique rules. First of all, JS is case sensitive. This means that declaring a variable like "myvar" and "myVar" will make those two variables be completely separate from each other. Now going with the syntax, JS runs the code from script files/elements from top to bottom. You can use linebreaks to separate different statements, or use semicolons to put them on a single line. A good practice is to declare statements on different lines and use semicolons. Finally, comments on js are like most languages; you can define a single line comment using "//" and multiline comments using "/**/".
