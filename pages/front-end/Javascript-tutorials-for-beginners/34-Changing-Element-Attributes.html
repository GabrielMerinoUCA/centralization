XXXIV. Changing Element Attributes

As we know, html elements have attributes, and such attributes can be managed using Javascript.

index.html



<body>

     <main id="main-id" class="main-content">

        . . .

     </main>

</body>


test.js



let myMainTag = document.getElementById("main-id");


/* The "getAttribute()" method returns the contents of whatever attribute you pass from the selected element */

console.log( myMainTag.getAttribute( "class" ) );// Prints "main-content"


/* The "setAttribute" does what it's name implies
arg1: attribute you want to change
arg2: new value to assign to such attribute
*/

myMainTag. setAttribute( "class", "navbar" ); // Changes main tag class value and therefore style

/*Also, if the attribute doesn't exist, it will create it*/


/* You can see the class attribute value or change it using this attribute */

myMainTag.className = "main-content";


Asides from this attributes and methods, there are many more, like href and src which return the complete path (not the value), and style which returns the whole style used by the element as an object.
