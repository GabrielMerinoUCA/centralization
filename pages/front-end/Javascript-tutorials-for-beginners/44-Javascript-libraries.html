XLIV. Javascript Libraries

Javascript libraries can bring some additional functionalities to your webpage. There are many of them and for this example, we'll check how to install one of them, JQuery. However, I do not recommend learning such library as it was originally created with the intention to bring consistency to the weirdness that old Javascript had, and of course, JQuery will make your page load slower.

Step 1: add the script tag

index.html



<body>


    . . .

     <!-- You add the script that references an external server. Always put your libraries before any other script that might depend on them -->

     <script src= "https://code.jquery.com/jquery-3.7.0.slim.min.js" integrity= "sha256-tG5mcZUtJsZvyKAxYLVXrmjKBVLd6VpVccqz/r4ypFE=" crossorigin= "anonymous"> </script>

<body>
