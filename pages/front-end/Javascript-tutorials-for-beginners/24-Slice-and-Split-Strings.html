XIV. Slice and Split Strings

24.1. Slice

This essentially is going to be a continuation to the previous chapter.

test.js



let str = "Hello world";

/*
This function allows you to extract a segment of any string
arg1: The first character's index from the slice of string you want to extract
arg2: The ending character's position from the slice of string you want to extract, yes, it's no longer the index for some reason
*/

let str2 = str. slice(2, 8);

console.log(str2); // Prints "llo wo"

If you don't pass a second argument, it will slice from the first arg, till the end of the string.

24.2. Split

The split method allows you to split a string into multiple strings. You define when you want to generate the split using a character or string. Once the character is found in the original string, a new string will be created and added to a string array. (More on arrays on the next chapter)

test.js



let str = "ham, peperoni, chicken, meat, fish, pork";

let strArray = str. split(", "); //split when comma and space found


console.log(strArray);

/* Result: ["ham", "peperoni", "chicken", "meat", "fish", "pork"] */
