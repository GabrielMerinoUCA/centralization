XI. If Statements

11.1. Basic expressions

There are different expressions that depending on the logic behind them will return a Boolean value. Before getting into them, here's how if statements are used in Javascript, as you can see, nothing out of the ordinary. I don't have to explain to you how conditions work, do I?

test.js



var youLikeMeat = true;

if(youLikeMeat) {
    console.log( "You like meat" );
}


Now into the expressions, first we have the greater than and lesser than expression. Again very self explanatory.

test.js



if(7 > 5) {
    console.log( "Indeed, 7 is greater than 5" );
}

if(7 < 5) {
    console.log( "This is impossible" ); // Of course, if false, won't execute
}


Then we have the following expression, the equals to expression.

test.js



if(10 == 10) {
    console.log( "It appears that 10 is equal to 10" );
}

11.2. Else statement

Javascript of course also has the else statement, which is used when the condition of an if statement is not true.

test.js



if(9 == 10) {
    console.log( "This is impossible" );
}

else {
    console.log( "It seems the condition is not true" );
}
