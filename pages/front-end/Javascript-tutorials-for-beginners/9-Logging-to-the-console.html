IX. Logging to the console

9.1. Document and Console

On your browser, there are two important things you have to know when a page is loaded, first that the page loaded is known as the document, and that you can execute JS code on the browser for the specific page using a console.

Why is this important? Because of the document and console built-in Javascript objects. Javascript has a lot of prebuilt stuff such as this objects which make some tasks easy to perform. We won't focus much on this objects for now, but just know that the document object is used to interact directly with the "document" and the console object is used to show data on the console, very useful for debugging.

9.2. write and log methods

We well get more in depth with the document and console object in later chapters for know we will talk about to methods used to log info in our pages from scripts, document.write(), and console.log().

/* This method will overwrite the whole document content, unless it has been used before, in which case, it will just add it at the end of the document */

document.write("Hello");




/* This method will show whatever you pass as argument on the browser's page console */

console.log("Hello");
