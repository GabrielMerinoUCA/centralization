Break & Continue

Adding to the stock of default language features that Javascript has, we have the continue and break keywords. We already know them, and yes, they work like on any other language.

test.js



//Stops at 5

for(var i = 0; i < 10; i++) {
    console.log( i );
    if(i === 5) {
        break;
    }
}


// Won't print 3 and 7

for(var i = 0; i < 10; i++) {
    if(i === 3 || i === 7) {
        continue;
    }
    console.log( i );
}
