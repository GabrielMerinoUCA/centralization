XXVIII. THIS Keyword

"this" is a OOP concept used to refer to the current object we're in. Javascript works in this same old fashioned way as it should

test.js



let myCar = {
    topSpeed: 100,
    driver: "John",
    whatThis: function() {
         console.log(this); // Prints current object info
    }
     getDriver: function() {
         return this.driver;
    }
 }


myCar.whatThis();


/* "this" is particularly useful on JS due to JS's global scope properties */

console.log( myCar.getDriver() );
