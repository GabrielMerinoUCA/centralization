XLII. Accessing Form Elements

42.1. Accessing Input data

Javascript can be used for form validation on the client side. To achieve this, we can access the form using a specific attribute usually used on forms

index.html



<body>

     <form name="form1">

          <input type="text" name="name" /><br>

          <input type="checkbox" name="biking" value="biking"/> Biking<br>

          <input type="checkbox" name="skiing" value="skiing"/> Skiing<br>

          <input type="checkbox" name="driving" value="driving"/> Driving<br>

          <select name="color">

               <option>Red</option>

               <option>Green</option>

               <option>Blue</option>

          </select>

          <input type="submit" name="submit value="Submit />

    </form>

</body>


test.js



/* The forms attribute contains an attribute collection of all the forms of the document. The name of such attributes is associated with the html name attribute you specify.*/

let myForm = document.forms.form1;


/* And the form itself works the same way but for all its input children */

let textInput = myForm.name;


/* The value attribute returns the current value of the input. In the case of the text input, it returns the value that the user inputted */

console.log( textInput.value );


/* This also applies to all inputs */

console.log( myForm.color.value );// Prints user selected color

42.2. onFocus & onBlur events

As stated in previous chapters, there are events everywhere. One of those events happens when you click on any input, selecting it. This triggers the "onFocus" event and allows us to run code when this happens.

index.html



(The same one)


test.js




. . .

textInput.onfocus = function() {
     textInput.style.border = "2px solid pink";
} // changes text input border style when selected


/* Of course, if we click off the input, the style will remain the same way. To return it back to it's previous state, you'll have to use "onBlur"*/

textInput.onfocus = function() {
     textInput.style.border = "none";
} // changes text input style when deselected


I want to add that there is a way to achieve the onfocus event on css. However, there is no onblur event. You could engineer something but it is simpler to use Javascript; and since you are already using Javascript, you might as well do everything on Javascript
