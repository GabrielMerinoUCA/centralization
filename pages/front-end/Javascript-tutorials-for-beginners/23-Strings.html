23.2. String object properties

Like many other languages, Javascript treats strings like objects, which means that there are a collective of methods and attributes available that we can use.

test.js



let x = "Hello";


/* returns the amount of characters of the string without including termination character */

x. length;


/* Make every character to upper case */

x. toUpperCase();


/* Make every character to lower case */

x. toLowerCase();


/* Find the first index of the first instance of the word sent on arguments (starts at 0)*/

x. indexOf( "lo" ); // returns 3

// if the string doesn't exist, it'll return -1


/* Javascript supports === operator for string comparison */

let a = "abc";

let b = "abc";


if(a === b) {
    console.log( "Both strings are the same" );
}
