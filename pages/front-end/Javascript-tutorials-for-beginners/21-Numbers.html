XXI. Numbers

Many languages have their prebuilt math libraries, some with more content than others. In the case of Javascript, wee have the Math object which provides some basic math functionalities.

test.js



let a = 5;

let b = 0.4;


// Rounding method

Math.round(a + b); // result is 5


// Always round to the lowest value method

Math.round(a + 0.9); // Result is 5


// Always round to hightest value method (except x.0)

Math.ceil(a + 0.1); // Result will be 6


// Find the biggest number from the args sent

Math.max(5, 3, 6, 9); // will return 9


// Also provides some common math numbers

console.log(Math.PI);
