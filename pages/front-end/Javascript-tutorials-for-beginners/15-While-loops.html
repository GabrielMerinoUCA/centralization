XV. While loops

While loops. A very important and common feature in programming languages, of course present in Javascript. We already know how they work but in summary, they will execute the same inner code as long as their condition is false.

test.js



var age = 5,


while(age < 18) {
    console.log( "you are underaged" );
    age++;
}

console.log( "You are now an adult" );
