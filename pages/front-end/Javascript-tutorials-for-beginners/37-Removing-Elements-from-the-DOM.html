XXXVII. Removing Elements from the DOM

It was to be expected that having the ability to add implies the existence of an ability to remove

index.html



<body>

    <ul>

          <li> Item 1 </li>

          <li> Item 2 </li>

          <li> Item 3 </li>

          <li> Item 4 </li>

    </ul>

</body>


test.js



let myListTag = document.getElementsByTagName( "body" )[0].getElementsByTagName( "ul" )[0];

let myListItem = myListTag.getElementsByTagName( "li" )[0];


/* Remember, myListItem is a reference so it is part of the "ul" */

myListTag.removeChild( myListItem ); // Removes list item "Item 1"

/* By the way, if you ever plan on reintegrating the element that you just removed, the "removeChild" method returns a new copy of the removed element */
