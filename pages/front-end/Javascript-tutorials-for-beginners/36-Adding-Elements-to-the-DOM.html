XXXVI. Adding Elements to the DOM

Managing the document from Javascript also includes adding new elements within the pre-declared objects.

index.html



<body>

    <ul>

          <li> Item 1 </li>

          <li> Item 2 </li>

          <li> Item 3 </li>

          <li> Item 4 </li>

    </ul>

</body>


test.js



/* Step 1: Create element to be added */

let myListItem = document.createElement( "li" ); // created new "li" element


/*Step 2: You can create another element that can be inserted into the previously created element */

let myAnchorTag = document.createElement( "a" ); // created new "a" element


/* Step 3: Retrieve the element parent (if necessary) to then retrieve the desired child element */

let myBodyTag = document.getElementByTagName( "body" )[0]; // Use id on html


/* Step 4: Get the desired element */

let myListTag = myBodyTag.getElementsByTagName( "ul" )[0];


/* Step 5: Append the created element to the desired element */

myListTag.appendChild( myListItem ); // The html ul now has 5 elements


/* Step 6: Since we already have a reference to the "li" we just created, we can add manipulate it and such changes will be reflected immediately on the webpage. */

myListItem.appendChild( myAnchorTag ); // The new html "li" now has an a tag


Alright that's one way of doing it however, you might have noticed that we can only insert elements at the bottom of the parent element. If we want to insert an element at any position, we have to use a different method

/* We can reuse the previously created "li" however since it's a reference, it will not create new list items
arg1: element to insert
arg2: element that will have the arg1 element inserted before it. */

myListTag.insertBefore( myListItem, myListTag.getElementsByTagName( "li" )[1]; )
