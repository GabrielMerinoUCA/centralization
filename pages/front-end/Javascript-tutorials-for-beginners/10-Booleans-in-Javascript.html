X. Booleans in Javascript


10.1. true and false

As stated previously, Javascript has the common found primitive data types as for other languages. The one we will be talking right now are booleans as the title suggests.

var trueVar = true;

var trueVar = false;


We'll get more in depth on some uses of this booleans. For now, this is their syntax

10.2. truthy and falsy

Unlike some other languages, Javascript supports what are so called truthy and falsy values. There are some different ways a value can result in truthy or falsy. In the case of numbers, any number equal to 0 will always be falsy and any other number (including negatives etc...) will result on it returning a truthy value

Javascript has a method to determine the truthy or falsy values so we know if they are true or false.

Boolean(-5);// return true

Boolean(0);// return false


You can also use this method on strings, a string with any character (at least one), will return truthy, and a string with no characters will return falsy

Boolean("Hello");// return true

Boolean("");// return false
