VII. Basic Mathematical Operators


7.1. The 5 most common operators

  • Assignment operator "="

var myVar = 4;


  • Subtraction operator "-"

var myVar1 = 10;

var myVar2 = myVar1 - 2; // Yes, myVar2 will be 8


  • Addition operator "+"

var myVar1 = 12;

var myVar2 = myVar1 + 3; // myVar2 will be 15


  • Division operator "/"

var myVar1 = 27;

var myVar2 = myVar1 / 9; // myVar2 will be 3


  • Multiplication operator " * "

var myVar1 = 5;

var myVar2 = myVar1 * 5; // myVar2 will be 25


7.2. Some oddities of Javascript

In Javascript, string concatenation exists, but the odd thing about it is that you can concatenate a string with a number

var myVar = "hello" + 5; // myVar will be equal to "hello5"


Of course the "add" sign works both as concatenation and addition, but that's only the addition sign. Trying to use any other operator on the same context will return a NaN or "Not a Number". A NaN is returned whenever a mathematical operation returns an invalid number. So it can be invalid arithmetic operations (syntax), or impossible operations (mathematically impossible)

var myVar = "hello" / 5; // myVar will be "NaN"
