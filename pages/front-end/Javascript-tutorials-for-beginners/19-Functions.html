XIX. Functions

19.1. Void functions

We all know functions and Javascript is no stranger to them. To quickly recap what they do, they help organize code on blocks where similar code is executed.

test.js



// no type definition needed on parameters

function getAverage(a, b) {
    var average = (a + b) / 2;
    console.log( average );
}


// You can not pass args but the function will not work as intended

getAverage(7, 12);

// Additionally, sending extra args will be ignored

19.2. Return functions

Well of course, Javascript also supports functions that return values.

test.js



function getAverage(a, b) {
    var average = (a + b) / 2;
    return average;
}


var result = getAverage(7, 12);

console.log(result);
