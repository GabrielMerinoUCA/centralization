II. What is Javascript


2.1 introduction to capabilities


Javascript is a language used for manipulating the front end on web development by adding behavior and interactivity.

Javascript is a scripting language (shockers), it is intentionally limited compared to real programming languages (c++) so no back end interactivity with this language (by default) like sql connections. However this language is good for manipulating web pages aka front end


2.2. What is it really


Javascript is used on client side. This means that it runs on the computer of the client instead of the server of the provider. This allows some in real time interactions and dynamic changes to the layout of the page itself and it's controllers


WARNING: DO NOT USE JAVASCRIPT TO ADD CORE FUNCTIONALITY TO YOUR WEBSITE LIKE ADDING ITEMS TO A SHOPPING LIST, ONLY USE TO ENHANCE THE USER EXPERIENCE (unless you using node.js etc...)


Extras: The official name of Js is ECMAScript
