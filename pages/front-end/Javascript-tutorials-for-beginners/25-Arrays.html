XXV. Arrays

25.1. Arrays on Javascript

Arrays are very common characteristics from any programming language, and as you might expect, Javascript is no exception. The syntax is very conventional as you will see.

test.js



let myArray = [];


/* JS is a weird language, you can insert values on any index of an array, when the array itself is empty*/

myArray[1] = 25;


/* But that's not the only weird thing. On Javascript, you can have an array of different data types */

myArray[0] = "This makes sense";


/* Now, something cool about Javascript is that you don't need a loop to see all the values of the array. You can directly print them by just specifying the array itself */

console.log(myArray); // Print "Array ["this makes sense", 58]"

25.2. Array properties

Javascript has a set of default methods supported by arrays like some other languages do.

test.js



// Show array length

let myArray = [20, 10, 5, 9];


console.log(myArray.length); // Prints 4


// Sort ascending

console.log(myArray.sort()); // Prints "Array [5, 9, 10, 20]"
