XXXV. Changing CSS Style

Just like you can manage the html elements of a document, you can also manage their style. However, you can not modify the css stylesheet itself, so any changes you do will be "local".

index.html



<body>

    <h1>

        . . .

    </h1>

</body>


test.js



let myHeaderTag = document.getElementsByTagName( "h1" )[0];


// you can change css properties using the style attribute

myHeaderTag.style.marginTop = "10px";

/* So, two things to note. First, the css properties are using camel case. Second, all the "style" attributes are type string (unless) */
